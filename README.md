# Ping Pong Game (Python)
A fully interactive two-player Ping Pong game built using Python and Turtle Graphics, designed with object-oriented programming and real-time animation control.

## Features
Real-time ball movement with adjustable speed for smooth gameplay.

### Dynamic collision detection with:
- Accurate ball reflection logic for realistic direction changes.
- Two-player paddle controls using keyboard event listeners
- Score tracking system with separate left and right player scores
- Automatic ball reset when a point is scored

### Clean separation of game logic using classes:
- Ball for movement and physics.
- Paddle for player controls.
- Score for score display and updates
- Manual screen refresh using screen.tracer(0) and screen.update() to eliminate flickering and control animations

## Technologies Used
- Python
- Turtle Graphics
- Object-Oriented Programming (OOP)

## Controls
- Right Paddle: Up / Down arrow keys
- Left Paddle: W / S keys
